Автоматическая авторизация

Полностью автоматический вход в аккаунты. Поддержка AdsPower и Vision Browser. Улучшенная работа с браузерами.

API Чекер

Новый чекер на API. Сортировка по статусам. Кнопка "Обновить статусы" для профилей с 0 просмотров. Общая статистика после действий.
